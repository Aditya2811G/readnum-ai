# Readnum AI  
AI-Powered Document Understanding & Insight Generation System

Readnum AI is a lightweight AI-driven web application designed to extract structured insights from unstructured text inputs. The system focuses on intelligent prompt engineering, response parsing, and modular workflow design to automate document understanding.

---

Problem Statement

Manual extraction of meaningful insights from large unstructured text data is time-consuming and inefficient.  
Readnum AI automates this process by leveraging AI-driven workflows to generate structured, actionable outputs.

---

Key Features

- AI-powered document summarization  
- Structured insight generation from free-form text  
- Prompt-engineered AI requests for controlled output formatting  
- Response parsing and structured display logic  
- Modular frontend architecture designed for backend scalability  

---

Tech Stack

- **Frontend:** HTML, CSS, Vanilla JavaScript  
- **AI Integration:** Google Gemini API  
- **Tools:** Git, VS Code  

---

System Architecture

User Input  
→ Prompt Structuring Layer  
→ Gemini API Request  
→ AI Response Parsing  
→ Structured Insight Rendering  

The application is designed with a clear separation between:
- Input handling
- Prompt engineering logic
- API communication
- Response processing
- UI rendering
  
Engineering Focus

This project demonstrates:

- Understanding of prompt engineering and output control  
- Structured workflow automation  
- Frontend system design with scalability considerations
