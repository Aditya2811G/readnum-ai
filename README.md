Readnum AI

AI-powered document processing and structured insight generation tool.

Problem Statement
Manual extraction of structured insights from unstructured text is time-consuming. Readnum AI automates this process using AI-driven workflows.

Features
- AI-based summarization
- Structured output generation
- Prompt engineered AI responses
- Modular frontend logic

Tech Stack
- JavaScript (Vanilla)
- HTML/CSS
- Google Gemini API

System Design
- User input → Structured Prompt → AI API → Parsed Output → UI Rendering

Future Improvements
- Node.js backend for secure API handling
- Database integration
- React-based frontend for scalability
